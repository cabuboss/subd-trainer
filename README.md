# 🗄️ Тренажёр СУБД

Интерактивная подготовка к экзамену по СУБД (MSSQL / T-SQL).

**Live:** https://cabuboss.github.io/subd-trainer

## Возможности
- 🎲 **Случайный режим** — все 354 вопроса в случайном порядке
- 📇 **Карточки** — режим flashcards для быстрого повторения
- 📚 **По разделам** — автоматическая классификация: SELECT, DDL, DML, Транзакции, Триггеры, Процедуры, Индексы, Нормализация, T-SQL функции и др.
- 🏁 **Марафон** — все вопросы по порядку без перерывов

## Управление
- `1`-`9` — выбор ответа
- `Enter` / `Space` / `→` — следующий вопрос
- `←` — назад (в карточках)

## Источник
Квиз "Субд" автор Санжар Бейбитов, 354 вопроса, материал для подготовки к экзамену.
